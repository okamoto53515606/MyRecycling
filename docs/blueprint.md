# **App Name**: MyRecycling

## Core Features:

- Hello World表示: "Hello, World!"というテキストを表示する。
- CSSファイルなし: 外部CSSファイルを一切使用しない。
- Tailwind CSS削除: package.jsonからtailwindcssおよびtailwind関連のパッケージを削除する
- 未使用モジュール削除: package.jsonから未使用のnpmモジュールを削除する。
- 最速・最軽量化: npmモジュールやソースコードを極力減らす。
- blueprint日本語: Nextjsのblueprintを日本語で記述する。

## Style Guidelines:

- 背景色: 白 (#FFFFFF)
- フォント: 'Inter' (sans-serif)。画面上のテキスト表示に使用。
- レイアウト: 最小限のレイアウトを使用し、中央に"Hello, World!"と表示する。
